Soal 4

Kita dapat mencari jarak kedua pusat lingkaran sebagai berikut:
d² = s² - (r1 + r2)²

s² = d² + (r1 + r2)²

s = 2√d2 + (r1 + r2)2
s = 2√242 + (6,5 + 3,5)2
s = 2√242 + (10)2
s = 2√576 + 100
s = 2√676
s = 26

Jarak titik pusat kedua lingkaran tersebut adalah 26 cm.
∴ Pilihan yang sesuai adalah (D) 26 cm

Soal 5

Diketahui panjang garis singgung persekutuan luar lingkaran dan jarak kedua pusat lingkaran, untuk mencari besar r2 dapat kita cari sebagai berikut:
I² = s² - (r1 - r2)²
20² = 25² - (18 - r2)²
400 = 625 - (324 - 36r2 + r2²)
400 = 624 - 324 + 36r2 - r2²
400 = 301 + 36r2 - r2²
400 - 301 - 36r2 + r2² = 0
99 - 36r2 + r2² = 0
(r2 - 3)(r2 - 33)=0
r2 = 3 atau r2 = 33
∴ Pilihan yang sesuai adalah (C) 3 cm

Soal 6

Pada segi empat tali buusr besar dari sua sudut yang berhadapan adalah 180°, maka dapat kita cari nilai x terlebih dahulu sebagai berikut:
∠ADC + ∠ABC = 180°
2x + 4° + 3x - 14° = 180°
5x - 10° = 180°
5x = 180° + 10°
5x = 190°
x = 190° : 5
x = 38°
∠ADC = 2x + 4°
∠ADC = 2(38)° + 4
∠ADC = 76° + 4°
∠ADC = 80°
∴ Pilihan yang sesuai adalah (B) 80°

Soal 7

Diketahui panjang tali busur adalah 24 cm dan panjang apotema tali busur adalah 16 cm, maka panjang jari-jari lingkaran dapat dicari sebagai berikut:
Panjang jari-jari2 = (panjang tali busur : 2)2 + (apotema)2
Panjang jari-jari2 = (24 : 2)2 + 162
Panjang jari-jari2 = 122 + 162
panjang jari-jari2 = 144 + 196
Panjang jari-jari2 = 400
Panjang jari-jari = 20
∴ Pilihan yang sesuai adalah (D) 20 cm

Soal 10

Pertama kita akan mencari panjang jari-jari lingkaran C, sebagai berikut:
   d² = s² - (r1 + r2)²
 12² = 13² - (r1 - 3)²
144 = 169 - (r1² - 6r1 + 9)
144 = 169 - 9 + 6r1 - r1²
144 = 160 + 6r1 - r1²
    0 = - 144 + 160 + 6r1 - r1²
    0 = 144 - 160 - 6r1 + r1²
    0 = - 16 - 6r1 + r1²
       r1² - 6r1 - 16 = 0
    (r1 + 8)(r1 - 2) = 0
   r1 = - 8 atau r1 = 2

Maka didapatkan r1 adalah 2 cm. Selanjutnya kita akan mencari perbandingan luas lingkaran C dan D sebagai berikut:

Luas lingkaran C : Luas lingkaran D
                      πr² : πr²
                 π × 2² : π × 3²
            π × 2 × 2 : π × 3 × 3
                  π × 4 : π × 9

                        4 : 9

∴ Pilihan yang sesuai adalah (E) 4 : 9
